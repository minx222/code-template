# 代码模版

## 使用
### 创建基座
```
template create
```
### 创建子应用
```
template app
```


## 命令
### 发布
```sh
npm publish
```

### 小版本
```sh
npm version patch
```

### 次版本号
```sh
npm version minor
```

### 主版本号
```sh
npm version major
```
